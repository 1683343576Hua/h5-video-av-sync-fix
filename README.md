# H5视频音画同步修复工具 | Audio Delay Adjuster
专门解决网页H5视频「音频比画面快、画面比声音慢」的音画不同步问题，通过Web Audio API实现页面全局音频输出延迟，无需修改视频播放速度，无卡顿、无掉帧，支持毫秒级精度调节和智能偏移检测。

## ✨ 核心功能特性
✅ **毫秒级精度调节**：支持0.001秒（1毫秒）步进调节，0~10秒超大延迟范围，精准对齐音画
✅ **智能偏移检测**：一键自动计算音画延迟差值，小白也能快速完成同步对齐
✅ **三档阶梯调节**：粗调/中调/微调三档按钮，快速大范围定位+精细对齐
✅ **人性化交互面板**：可拖动悬浮控制面板，支持一键折叠收起，不遮挡视频画面
✅ **全屏快捷键支持**：全屏观影无需切出，键盘快捷键即可完成微调
✅ **配置自动记忆**：自动保存用户的延迟配置，刷新页面/重启浏览器自动恢复
✅ **全场景兼容**：支持所有原生H5视频网站、iframe内嵌视频、动态加载的视频
✅ **一键测试验证**：内置测试音功能，快速验证延迟调节是否生效

## 🚀 安装教程
### 前置要求
1.  浏览器已安装脚本管理器：推荐 [Tampermonkey](sslocal://flow/file_open?url=https%3A%2F%2Fwww.tampermonkey.net%2F&flow_extra=eyJsaW5rX3R5cGUiOiJjb2RlX2ludGVycHJldGVyIn0=)（兼容Chrome/Edge/Firefox/ Safari等所有主流浏览器）
2.  脚本管理器启用状态正常

### 安装方式
#### 方式一：Greasy Fork一键安装（推荐普通用户）
直接访问脚本Greasy Fork主页：`替换为你的Greasy Fork脚本地址`，点击「安装此脚本」即可。

#### 方式二：手动安装（开发者/自定义修改）
1.  复制本仓库内 `audio-delay-adjuster.user.js` 文件的完整代码
2.  打开浏览器油猴插件，点击「添加新脚本」
3.  清空默认模板代码，粘贴复制的完整脚本代码
4.  按 `Ctrl+S` 保存，启用脚本即可

## 📖 基础使用方法
1.  安装并启用脚本后，打开任意包含H5视频的网页
2.  播放视频，页面右上角会自动弹出调节控制面板
3.  核心操作逻辑：**音频比画面快 → 增加延迟值，让音频晚一点输出**
    - 拖动滑块无级调节延迟值
    - 点击三档按钮精准调节：±0.1s粗调、±0.01s中调、±0.001s微调
4.  调节实时生效，无需暂停/重启视频
5.  面板可按住顶部标题栏拖动位置，点击「—」可折叠收起，完全不遮挡画面

---

## 🎯 智能检测功能使用教程
针对小白用户，无需手动估算延迟，一键自动计算精准差值：
1.  播放视频，找到一个**音画同步点**（推荐短促的声音+画面，如枪响、拍手、人物说话的爆破音）
2.  点击面板上的「开始检测」按钮
3.  当**听到对应声音**时，立即点击「标记声音」按钮
4.  当**看到对应的画面动作**时，立即点击「标记画面」按钮
5.  脚本自动计算出精准的音画延迟差值，点击「应用延迟」即可一键完成对齐

## ⌨️ 全屏快捷键说明
全屏观影无需切出，直接用键盘完成调节：
| 快捷键 | 功能 |
|--------|------|
| 方向键 → | 增加0.001秒音频延迟 |
| 方向键 ← | 减少0.001秒音频延迟 |
| 空格键 | 一键重置延迟为0 |

## ❓ 常见问题
### Q1: 调节滑块没反应，延迟不生效？
A: 
1.  先点击播放视频，再点击「播放测试音」，能听到哔声说明音频引擎正常
2.  刷新页面，先打开脚本面板，再播放视频
3.  检查视频是否在无痕窗口/浏览器画中画模式，无痕窗口需要单独开启油猴权限

### Q2: 调节后视频没有声音了？
A: 点击「重置为0」按钮即可恢复正常声音；若仍无效，刷新页面重新播放视频即可。

### Q3: 提示「视频连接失败」怎么办？
A: 该网站使用了DRM数字版权加密保护，浏览器限制无法修改加密音频流，此为正常现象，本脚本无法突破版权保护限制。

### Q4: 面板挡住视频了怎么办？
A: 按住面板顶部的标题栏，可随意拖动到页面任意位置；点击「—」按钮可折叠成小方块，完全不遮挡画面。
